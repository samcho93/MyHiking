## 개발 환경
- 모바일 App
- 안드로이드 전용

## 기능
- 스마트폰의 이미지 검색하여 위치데이터 취득
- 대한민국의 지도 데이터를 로드
- 이미지의 위치데이터를 읽어서 지도 상의 산 주변에서 찍은 사진이 있는 지 검색.
- 산 주변에서 찍은 사진을 모두 취득
- 화면상에 보이는 산 정상 부근에 썸네일 형태의 이미지를 띄움.
- 다수개의 사진은 다수개를 표현할 수 있는 썸네일 겹침을 활용
- 썸네일을 클릭하면, 산 주변의 사진을 볼 수 있게 미리보기가 있는 리스트를 보여줌
- 사진을 선택하면 전체화면으로 보여주며, 뒤로 가기나 [X] 버튼으로 리스트로 돌아감
- 리스트에서 뒤로가기나 [X] 버튼으로 지도 화면으로 돌아감
- 지도는 네이버 지도 API를 활용함
- 별도의 이미지 저장 공간을 만들지 않고, 스마트폰의 사진을 검색하여 해당 리스트를 가지고 있어야 함.
- 대한민국을 벗어난 산은 구글 이미지 API를 활용하며, 사용자의 선택에 의해 네이버와 구글 지도 선택을 할 수 있음.
- 네이버 지도 선택은 국내산만, 구글 지도 선택은 전세계 산 모두
- 네이버 지도 선택은 국내만 보임. 구글 지도는 사진이 있는 위치의 영역만 보이도록 함.
- 앱을 실행하면 첫페이지는 네이버 지도에 산위 위치가 표시되며, 근처에서 찍은 사진이 있으면 하이라이트하여 사진 썸네일을 보여줌.
