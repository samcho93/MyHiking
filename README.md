# MyHiking

**MyHiking**은 스마트폰에 저장된 등산 사진의 GPS 데이터를 분석하여 한국의 산과 자동으로 매칭해주는 Android 앱입니다. 5,897개의 한국 산 데이터베이스와 거리+고도 기반 스코어링 알고리즘을 사용하여 사진이 촬영된 산을 정확하게 식별하고, 지도 위에 썸네일 마커로 표시합니다.

---

## 주요 기능

- **자동 사진-산 매칭**: GPS 좌표와 고도 정보를 분석하여 가장 가까운 산에 자동 매칭
- **듀얼 지도 지원**: 네이버 지도 (한국) / Google Maps (해외) 실시간 전환
- **100대 명산 시각 구분**: 산림청 + 블랙야크 100대 명산을 금색 테두리(⭐)로 강조 표시
- **썸네일 마커**: 실제 사진 썸네일을 지도 마커로 표시 (복수 사진 시 스택 마커)
- **사진 재배정**: 잘못 매칭된 사진을 다른 산으로 수동 이동
- **사진 제외**: 등산과 무관한 사진 제외 (개별/일괄)
- **위치 없는 사진 추가**: GPS 정보가 없는 사진을 특정 산에 수동 추가
- **해외 산 클러스터링**: 한국 외 지역 사진을 자동 그룹화 + 역지오코딩
- **증분 스캔**: 이전 스캔 결과를 캐시하여 빠른 재시작
- **위치 보기 모드**: 산 단위/GPS 좌표 단위 표시 전환

## 매칭 알고리즘

### 사진-산 매칭 프로세스

```
사진 GPS 좌표 (위도, 경도, 고도)
        │
        ▼
┌─ 1단계: Pre-Filter ─────────────────┐
│  위도/경도 차이 < 0.011° (~1.1km)    │
│  빠른 후보 산 필터링                   │
└──────────────┬──────────────────────┘
               ▼
┌─ 2단계: Haversine 거리 계산 ─────────┐
│  반경 1,000m 이내 산 탐색              │
│  지구 곡률 반영 (R=6,371km)           │
└──────────────┬──────────────────────┘
               ▼
┌─ 3단계: 고도 검증 ──────────────────┐
│  사진 고도 >= 산 고도의 20%            │
│  사진 고도 <= 산 고도 + 500m          │
│  최소 30m 이상 (지면 사진 제외)         │
└──────────────┬──────────────────────┘
               ▼
┌─ 4단계: 스코어 계산 ─────────────────┐
│  score = 수평거리 + (고도차 × 3.0)     │
│  고도 100m 차이 = 수평 300m 페널티      │
│  최저 점수 = 최적 매칭                  │
└──────────────┬──────────────────────┘
               ▼
         최종 매칭 결과
```

### 주요 상수

| 상수 | 값 | 설명 |
|------|------|------|
| `DEFAULT_RADIUS_METERS` | 1,000m | 매칭 검색 반경 |
| `LAT_LNG_THRESHOLD` | 0.011° | Pre-filter 임계값 (~1.1km) |
| `MIN_ALTITUDE_RATIO` | 0.2 | 사진 고도 >= 산 고도의 20% |
| `ALTITUDE_UPPER_TOLERANCE` | 500m | GPS 오차 허용 범위 |
| `MIN_ALTITUDE_THRESHOLD` | 30m | 최소 고도 (지면 사진 제외) |
| `ALTITUDE_WEIGHT` | 3.0 | 고도 가중치 (1m 고도차 = 3m 수평거리) |
| `FOREIGN_MIN_ALTITUDE` | 100m | 해외 사진 최소 고도 |
| `CLUSTER_RADIUS_METERS` | 1,000m | 해외 사진 클러스터링 반경 |

## 사용법

### 1. 설치 및 최초 실행

```
1. APK 설치 후 앱 실행
2. 권한 허용:
   - 사진/미디어 접근 권한
   - 위치 정보 접근 권한 (사진 GPS 데이터 읽기용)
3. 자동으로 디바이스 사진 스캔 시작
4. GPS 태그가 있는 사진을 한국의 산과 매칭
5. 지도에 썸네일 마커로 결과 표시
```

### 2. 지도 조작

| 동작 | 기능 |
|------|------|
| **마커 탭** | 해당 산의 사진 목록 표시 |
| **마커 길게 누르기** | 3가지 옵션: 사진 이동 / 전체 제외 / 사진 추가 |
| **빈 지도 길게 누르기** | 근처 산 목록에서 선택하여 사진 추가 |
| **FAB 버튼** | 네이버 지도 / Google 지도 전환 |
| **[위치로 보기] 체크박스** | 산 단위 / GPS 좌표 단위 표시 전환 |

### 3. 사진 관리

| 기능 | 조작 방법 |
|------|----------|
| **사진 보기** | 사진 목록에서 썸네일 탭 |
| **사진 재배정** | 사진 목록에서 썸네일 길게 누르기 → 재배정 선택 |
| **사진 제외** | 사진 목록에서 썸네일 길게 누르기 → 제외 선택 |
| **일괄 제외** | 마커 길게 누르기 → 전체 사진 제외 |
| **사진 추가** | 사진 목록의 [+] 버튼 또는 마커/빈 지도 길게 누르기 |

### 4. 100대 명산 구분

산림청 및 블랙야크 100대 명산은 다음과 같이 시각적으로 구분됩니다:

- **금색 테두리** (#FFD700) 마커 프레임
- **주황색 뱃지** (#FF8F00) 사진 수 표시
- **⭐ 접두사** 산 이름 앞에 별표 표시
- **주황색 캡션** (#E65100) 네이버 지도 캡션 색상
